# Goal Tracker API
Backend API of MERN stack Goal Tracker App

## DEV ENV
1. Node js
2. Express js

# How to run app:
1. install node js
2. install packages using  
`npm install`
3. create env variables
4. run file using command  
`node filename.js`

## Required ENV variables:
1. NODE_ENV
2. PORT
3. MONGO_URI
